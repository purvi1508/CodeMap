# CodeMap 

> AI-Powered Code Architecture Analysis and Documentation Generator

CodeMap analyzes your codebase and automatically generates architecture diagrams, documentation, and insights using AI. Perfect for understanding legacy code, onboarding new developers, or documenting your project.
